# EscapeFromSZOK
2020.1.31\
大学の授業で作成した迷路ゲームです。

![スクリーンショット (2)](https://user-images.githubusercontent.com/56964484/119644366-bc914000-be57-11eb-8a01-f9e6c49093f6.png)

## start
```
$ javac *.java
$ java MapGame
```

## 操作
- 上下左右キーで移動
- 単位を３つ拾うと右下の扉が開く
- 攻撃ボタンでレポート提出

## 動作環境
java/javac(java-8-openjdk-amd64), javafxが必要
